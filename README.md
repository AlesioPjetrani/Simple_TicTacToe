# Simple_TicTacToe

This is a very simple implementation of the game 'Tic Tac Toe' in C between two local players in the same machine. 

(This is still a work in progress. The logic of the project is still not there, but the structure is being set up)
## Pre-Requisites
- `clang` compiler
- `make` build automation tool
- `git` version control system
- `valgrind` (optional, for memory leak detection and error tracking)
- `build-essential` package on Debian-based systems

If all of these tools are not already installed on your system (with the exception of `valgrind`), please follow the instructions below to set them up.

## Installating pre-requisites
- On Debian-based systems (like Ubuntu), you can install the required tools using:
```bash
  sudo apt update
  sudo apt install clang make git valgrind build-essential
```

## How to run

- Copy this repository in your local machine using `git clone` and go inside the cloned repository folder.
- There is a provided ```Makefile``` in the repository. Once you run `make` or `make help` it will display a list of available commands to compile and run the project.
